## Journo - Your News Hunting App

#### Install Journo Server
    $ cd journo/install
    $ ./install.sh

#### Start Request Server
    $ cd journo/server
    $ ./RequestServer.sh

#### Start Task Server
    $ cd journo/server
    $ ./TaskServer.sh
    
#### Start Tus Server
    $ cd journo/server
    $ ./TusServer.sh
